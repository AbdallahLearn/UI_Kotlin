# 📱 Interactive Profile Card App  

## 📌 Overview  
This is a simple **Jetpack Compose** application that displays an **interactive profile card**. The app includes a **profile picture, name, bio, and a Follow button** that toggles between "Follow" and "Following" while updating the follower count.  

## 🎯 Features  
✔ **Profile Card UI** with an image, name, and bio.  
✔ **Follow Button** to toggle follow state.  
✔ **Follower Count Update** when the button is clicked.  
✔ **Jetpack Compose UI components** like **Column, Row, Box, Image, Text, and Button**.  

## 🛠️ Tech Stack  
- **Kotlin**  
- **Jetpack Compose**  
- **Android Studio**

## 🚀 Getting Started  

### 1️⃣ Prerequisites  
Make sure you have the following installed:  
- **Android Studio** (latest version)  
- **Kotlin**  
- **Jetpack Compose Setup**  

### 2️⃣ Clone the Repository  
```sh
git clone https://github.com/your-username/interactive-profile-card.git
cd interactive-profile-card
```

### 3️⃣ Open in Android Studio  
1. Open **Android Studio**  
2. Click **"Open an Existing Project"**  
3. Select the cloned repository folder  

### 4️⃣ Run the App  
- **Connect an Android Emulator or Device**  
- Click **"Run" (▶️)** in Android Studio
